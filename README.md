# Personal Finance Tracker

Personal Finance Tracker is a simple, privacy-focused application designed to help users monitor income, expenses, categorize transactions, and visualize spending patterns for better financial management.

Built for ease of use, it allows self-hosting to keep sensitive data local. The project emphasizes straightforward tracking without unnecessary complexity.

## Features

- Add, edit, and delete income and expense transactions.
- Categorize transactions (e.g., food, utilities, salary).
- Generate basic reports on savings, spending by category, and net balance.
- Simple visualizations like pie charts for expense breakdowns.
- Data persistence using local storage or a lightweight database.

## Prerequisites

- Python .
- Install dependencies via `pip install -r requirements.txt`.

## Quick Start

1. Clone the repository:
   ```
   git clone https://github.com/SOMILSRIVASTAV18/Personal-Finance-Tracker.git
   cd Personal-Finance-Tracker
   ```
2. Set up the environment (Python example):
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python app.py  # or streamlit run app.py if using Streamlit
   ```
4. Open your browser to `http://localhost:5000` .

## Usage

- **Add Transaction**: Enter amount, category, date, and type (income/expense).
- **View Dashboard**: See summary charts and recent activity.
- **Reports**: Filter by date range or category for insights.
- Data saves automatically; export to CSV for backups.
